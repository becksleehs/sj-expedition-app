# 승주 원정대 v2.1 · RPG World & Bus Quiz

게임 콘셉트 이미지처럼 보이도록 메인 HUD와 퀘스트 기능을 강화하고,
버스 이동시간에 학생들이 즐길 수 있는 부산·울릉도·독도 퀴즈를 추가한 버전입니다.

## 핵심 변경
- 캐릭터 프로필, 레벨, 별, EXP가 표시되는 게임 HUD
- 일일퀘스트·공지사항·이벤트 바로가기
- 버스 원정 퀴즈 전용 화면
- 부산 롯데월드·울릉도·독도 각 10문제, 총 30문제
- 세 지역 혼합 랜덤 퀴즈
- 정답·연속 정답에 따른 EXP와 별 획득
- 지역 퀴즈 8문제 이상 정답 시 지역 배지 획득
- 퀴즈 EXP가 선택 학생의 기존 RPG 점수에 자동 합산
- 로컬 저장 및 서비스워커 캐시로 이동 중 오프라인 사용 지원

## 퀴즈 자료 기반
- 롯데월드 어드벤처 부산 공식 홈페이지
- 부산광역시 기본현황 및 부산 소개
- 울릉군 공식 홈페이지
- 대한민국 외교부 독도 자료
- 동북아역사재단 독도 교육 자료

## 주의
현재 데이터는 기기별 localStorage에 저장됩니다.
따라서 여러 학생 기기의 점수가 교사 PC에 자동으로 실시간 합쳐지지는 않습니다.
실시간 통합 랭킹은 Firebase 또는 Supabase 연동 단계에서 구현해야 합니다.
