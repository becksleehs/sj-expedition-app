승주 원정대 v1.0 저장소 업로드 안내

1. 다운로드한 sj-expedition-v1.0.zip의 압축을 풉니다.
2. GitHub Desktop을 실행하고 sj-expedition-app 저장소를 선택합니다.
3. 상단 Repository → Show in Explorer를 누릅니다.
4. 현재 저장소 파일을 다른 폴더에 백업합니다.
5. 압축을 푼 sj-expedition-v1.0 폴더 '안'의 파일을 저장소 폴더에 모두 복사합니다.
   ※ sj-expedition-v1.0 폴더 자체를 저장소 안에 넣으면 안 됩니다.
6. 같은 이름의 파일을 덮어씁니다.
7. GitHub Desktop Summary:
   Upgrade SJ Expedition to v1.0
8. Commit to main → Push origin
9. GitHub Actions에서 맨 위 실행이 초록색 체크가 될 때까지 기다립니다.
10. 주소 접속 후 Ctrl+F5:
    https://becksleehs.github.io/sj-expedition-app/

- 관리자 초기 비밀번호 1004는 배포 후 관리자 화면에서 변경
